# Course-Scheduling-System

#### 本项目旨在开发一个针对高校的排课系统，通过智能化的排课算法，实现资源的合理调度，同时解决排课冲突。本项目将以直观、美观的用户界面，为高校提供高效、便捷的课程调度解决方案。

#### 本项目将实现：

#### **1.** 多维约束配置，包括硬约束(如，同一教师同一时间只能在一间教室)和软约束(如，同一门课的两节课之间至少间隔一天)，系统允许管理员为不同的软约束设置权重，算法将根据权重寻找最优解。

#### **2.** 基于深度强化学习的调度算法内核。

#### **3.** 课表微调，允许管理员对算法给出的结果中某节课的排课位置进行调整。

#### **4.** 应对突发状况，允许发生突发状况(如，教师请假、教室装修)时，管理员对课表进行局部重排。

#### **5**.导出基于排课结果的分析报告，包括教室利用率、教师工作强度等方面。
